Tagging:

  • HTML is structured so that everything happens between two tags (ex: "tag" STUFF "end tag")
  • The beginning of a tag looks like this <tag> while the end looks like </tag>
  • It is possible to have a single tag. Single tag elements are elements that insert something into a page

Document format:

  • All HTML documents start with <html>
  • Next is <head> which supplies information to the browser and web server (it also labels the whole HTML document)
    • Inside of the <head> section should be <title> which is the text that appears in the browser bar
  • Next is <body> which is where any content that appears on the web page should be added

Page Redirection:

  • The command <meta http-equiv="refresh" content="15; url=http://www.w3.org/"> redirects the website to w3.org after 15 seconds
  • The command <meta http-equiv="refresh" content="1800"> refreshes the browser every 30 minutes

Links:

  • Intradocument links:
    • The command <a id="top"></a> creates a location within a page for direct access from other links.
    • The command <a href="#top">Back to top</a> creates a link that returns to top's location. (#) indicates that the location is on the same page and not a seperate page.
